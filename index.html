Scaffold CMS, taking the headaches out of content mangement.

Scaffold CMS allows you to handle all of your CMS website content within the Scaffold CMS app. All content is then pulled in via simple API calls using your favorite tech stack. No more having to worry about security concerns of other self hosted content management systems which frees up your time to focus on what matters!

How Scaffold CMS Works

Create a new property

Properties represent an individual website. Once a property is created you can begin to define the pages, fields, and post types you'll want to use in your website.

Define your Data

Once you've created a property, the next step is to define the pages, fields, and post types you want to use on your pages. Scaffold will generate forms you can use to populate the data for your website.

Build your Content

Once your data is devined Scaffold will generate beautiful forms that you and your clients can use to populate your sites data. Couldn't be easier!

Grab your data

The Scaffoldcms.io API is simple, restful and JSON only. Simply attach your property token to each API call you make and use it to populate the content of your site.
